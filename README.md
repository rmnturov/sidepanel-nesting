# Side Panel Layering App

Приложение с наслаивающимися боковыми панелями в стиле iOS.

## Особенности

- Боковые панели открываются справа с плавной анимацией iOS-стиля
- Поддержка до 2 уровней вложенности панелей
- Единое затемнение фона (backdrop) под всеми панелями
- Отступы 8px от краев экрана с блокировкой событий мыши
- Ширина панели: 640px, высота: на весь экран

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build
```

## Структура проекта

- `src/App.jsx` - основной компонент приложения
- `src/components/Backdrop.jsx` - компонент затемнения фона
- `src/components/SidePanel.jsx` - компонент боковой панели
- `src/App.css` - стили приложения

## Использование

1. Нажмите кнопку "Открыть панель 1" в центре экрана
2. Внутри первой панели нажмите "Открыть панель 2" для открытия второй панели
3. Клик по затемненному фону закрывает все панели
